### 📍 Why this Project?

This project is part of my exploration into high-impact visual interfaces. The goal was to create a **Premium Landing Page for Porsche**, focusing on a fluid and cinematic user experience.

Moving beyond a standard institutional site, the primary focus here was mastering animation libraries and DOM manipulation:
* **GSAP (GreenSock Animation Platform):** I implemented coordinated entry animations (staggering) for titles and visual elements, ensuring the user's first impression is both sophisticated and impactful.

* **Swiper.js Integration:** I utilized vertical navigation with coverflow and depth effects, providing seamless transitions between different car models.

* **Modular Script Architecture:** The project was structured using ES6 modules `(import/export)`, separating animation logic from the main execution for better maintainability.

* **UI/UX Design:** A heavy focus on custom typography (Porsche Font) and color contrasts that align with the brand's luxury identity.

### ⭐ How to use this project locally?

* To run this project on your machine, simply clone this repository. The project uses Vanilla JS and libraries via CDN, so no package manager installation is required.

```bash
git clone https://github.com/CaioLeal/Porsche-Landing-Page.git
```
Right after that, open the project folder in your IDE (VSCode) or navigate to the folder: ```index.html```
